# How to Add unDraw Illustrations

unDraw.co doesn't provide a public API for hotlinking. You need to download the SVGs and host them locally.

## Quick Setup:

1. **Go to https://undraw.co/illustrations**
2. **Search for illustrations** you want (e.g., "searching", "calendar", "team")
3. **Click on an illustration** to open it
4. **Click "Download"** and choose SVG format
5. **Save the SVG** to `public/illustrations/` folder with a descriptive name
6. **Update the code** to reference `/illustrations/filename.svg`

## Example:

If you download "searching.svg" and save it to `public/illustrations/searching.svg`, 
then in your code use: `src="/illustrations/searching.svg"`

## Recommended Illustrations for This Site:

## Color Customization:

When downloading from unDraw, you can customize the color to match your brand (primary color: #6366f1).
